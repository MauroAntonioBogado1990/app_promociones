# app_promociones
In this app you will can send to message with promotions for clients in Odoo


En estas carpetas estan presentes los desarrollos en flet , donde para probarlos recordar que se debe colocar el comando flet run nombredelarchivo.py, (previamente teniendo instalado flet: pip install flet[all])
como se especifican en los nombres de las carpetas estos desarrollos estan hechos para el caso de que se posea la ip del cliente, y para cuando no se la posee. El archivo que esta fuera de las carpetas 'app_promo.py', esta desarrollado mediante un botón que realiza un llamado a Odoo para verificar si existen notificaciones pero no sería el funcionamiento ideal.
